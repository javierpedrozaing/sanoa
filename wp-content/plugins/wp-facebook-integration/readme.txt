Integrate a Picture Gallery from your Facebook Page on the Fly!
 
== Description ==

<p>Add your Page ID  in the Settings Page.</p> 
<p>Use the shortcode [wpfbint_gallery] in any page or widget you like. </p>
<p>On the Fly you have a Gallery of your Latest Timeline Pictures</p>

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory and unzip, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WP Facebook Integration Menu Link or Settings link on plugins page to use the plugin.
4. Add your Facebook Page ID and use the shortcode on any of your pages to use your Gallery.

== Frequently Asked Questions ==

= Can I configure the Number of Pictures displayed in the Gallery? =
Yes, you define it from Settings Page.
